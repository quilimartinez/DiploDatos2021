Práctico 4: Aprendizaje Supervisado

Objetivo:
Sobre el modelo de regresión logística realizado en el practico anterior, se requiere profundizar el trabajo con el fin de obtener resultados de mayor calidad.

Metodología: trabajar sobre la ingeniería de features, el modelado y evaluación de las salidas de los modelos generados

1. Crear 5 nuevos features relacionando las varaibles del dataset
3. Utilizar un nuevo criterio de normalización de variables (logaritmo, raiz cuadrada, etc)
2. Utilizar 2 métodos de seleccion de variables (regresion forward, regresion backward, regresion stepwise)
4. Elegir el modelo mas performante

Entregas:

1. Breve informe (no más de 2 o 3 páginas) en el cual se explique qué detectaron en el set de datos, qué interpretación le dieron, alternativas de solución y propuesta.
2. El dataset con los cambios implementados en cuanto a curación y normalización.
